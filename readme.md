## Iprocure

A reusable procurement system.


### REST Api

#### Items

```
GET         /items          // index
POST        /items          // create
GET         /items/{item}   // show
PUT|PATCH   /items/{item}   // update
DELETE      /items/{item}   // destroy
```

#### Units

```

POST        /units          // create
GET         /units/{unit}   // show
```
